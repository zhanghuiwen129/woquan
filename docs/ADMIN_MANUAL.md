# 后台管理系统使用指南

## 系统概述

后台管理系统是社交应用的核心管理平台，提供全面的系统管理、用户管理、内容管理、数据分析等功能。本指南将详细介绍后台管理系统的各项功能和使用方法，帮助管理员快速掌握系统操作。

## 登录与访问

### 访问地址

后台管理系统默认访问地址：
```
http://your-domain.com/admin
```

### 登录账号

- **默认管理员账号**：admin
- **默认管理员密码**：admin123

> **安全提示**：首次登录后请立即修改管理员密码，确保系统安全。

## 系统布局

后台管理系统采用左侧固定导航栏 + 右侧内容区的布局结构：

1. **顶部导航栏**：显示系统名称、当前用户信息、通知、设置等
2. **左侧导航栏**：包含所有功能模块的导航菜单
3. **右侧内容区**：显示当前功能模块的详细内容
4. **底部状态栏**：显示系统版本、服务器状态等信息

## 核心功能模块

### 1. 系统管理

#### 1.1 系统设置

**功能说明**：管理网站基本配置、运营设置、安全设置等。

**子模块**：
- **基本设置**：网站名称、Logo、描述、版权信息等
- **网站设置**：网站域名、ICP备案、客服信息等
- **运营设置**：注册开关、审核设置、内容限制等
- **安全设置**：防刷设置、敏感词过滤、IP限制等
- **存储设置**：上传配置、存储方案、CDN设置等
- **邮箱设置**：SMTP配置、邮件模板等
- **通知设置**：系统通知、消息模板等
- **SEO设置**：网站标题、关键词、描述等

**使用指南**：
1. 进入「系统设置」模块
2. 选择需要修改的设置项
3. 填写相关配置
4. 点击「保存」按钮

#### 1.2 系统工具

**功能说明**：提供系统维护、数据库管理、缓存清理等工具。

**子模块**：
- **数据库维护**：数据库备份、恢复、优化

**使用指南**：
1. 进入「系统工具」模块
2. 选择需要使用的工具
3. 按照提示操作
4. 查看操作结果

#### 1.3 服务器管理

**功能说明**：管理服务器配置、监控服务器状态。

**子模块**：

## 操作指南

### 1. 用户管理操作

#### 添加用户
1. 进入「用户管理」→「用户列表」
2. 点击「添加」按钮
3. 填写用户信息
4. 点击「保存」按钮

#### 编辑用户
1. 进入「用户管理」→「用户列表」
2. 找到需要编辑的用户
3. 点击「编辑」按钮
4. 修改用户信息
5. 点击「保存」按钮

#### 禁用用户
1. 进入「用户管理」→「用户列表」
2. 找到需要禁用的用户
3. 点击「禁用」按钮
4. 确认操作

### 2. 内容管理操作

#### 发布动态
1. 进入「内容管理」→「动态管理」→「动态列表」
2. 点击「添加」按钮
3. 填写动态内容
4. 点击「发布」按钮

#### 审核动态
1. 进入「内容管理」→「动态管理」→「动态审核」
2. 找到需要审核的动态
3. 点击「通过」或「拒绝」按钮
4. 填写审核意见（可选）

#### 删除动态
1. 进入「内容管理」→「动态管理」→「动态列表」
2. 找到需要删除的动态
3. 点击「删除」按钮
4. 确认操作

### 3. 系统设置操作

#### 修改网站名称
1. 进入「系统管理」→「系统设置」→「基本设置」
2. 在「网站名称」字段中输入新的网站名称
3. 点击「保存」按钮

#### 配置存储方案
1. 进入「系统管理」→「系统设置」→「存储设置」
2. 选择存储方案（本地存储/云存储）

**解决方案**：
- 清理系统缓存

- 检查服务器资源使用情况

**问题**：数据显示不正确或丢失
**解决方案**：
- 检查数据库连接
- 恢复最近的数据库备份
- 联系技术支持

### 5. 安全问题

**问题**：系统可能存在安全风险
**解决方案**：
- 及时更新系统补丁
- 定期修改管理员密码

- 配置安全组和防火墙
