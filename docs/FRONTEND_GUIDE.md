# 前端主页功能说明文档

## 系统概述

前端主页是用户与社交应用交互的主要界面，提供丰富的社交功能和个性化体验。本文档详细介绍前端主页的各项功能和使用方法，帮助用户快速掌握系统操作。

## 访问与登录

### 访问地址

前端主页默认访问地址：
```
http://your-domain.com
```

### 登录方式

- **账号密码登录**：输入用户名和密码
- **手机验证码登录**：输入手机号和验证码
- **第三方登录**：微信、QQ、微博等

### 注册方式

- **手机号注册**：输入手机号、验证码、密码
- **邮箱注册**：输入邮箱、验证码、密码

## 页面布局

前端主页采用响应式设计，根据设备类型自动调整布局：

### PC端布局

1. **顶部导航栏**：网站logo、搜索框、消息通知、用户头像
2. **左侧边栏**：主要功能导航菜单
3. **中间内容区**：动态信息流、推荐内容
4. **右侧边栏**：热门话题、在线用户、推荐关注

### 移动端布局

1. **顶部导航栏**：网站logo、搜索图标、消息通知
2. **底部导航栏**：主页、发现、发布、消息、我的
3. **内容区**：根据当前功能显示相应内容

## 核心功能模块

### 1. 首页（动态信息流）

**功能说明**：展示用户关注的好友动态、热门动态、推荐内容。

**核心功能**：
- **动态浏览**：上下滑动浏览动态内容
- **动态发布**：点击发布按钮，支持文字、图片、视频发布
- **动态互动**：点赞、评论、分享动态
- **话题标签**：点击话题标签查看相关动态
- **地理位置**：查看动态发布位置

**操作指南**：
1. **浏览动态**：上下滑动屏幕
2. **发布动态**：点击底部或顶部的发布按钮
3. **点赞动态**：点击动态下方的点赞图标
4. **评论动态**：点击动态下方的评论图标，输入评论内容
5. **分享动态**：点击动态下方的分享图标，选择分享方式

### 2. 发现页

**功能说明**：发现热门内容、附近的人、兴趣话题等。

**核心功能**：
- **热门推荐**：根据热度推荐动态、用户、话题
- **附近的人**：基于地理位置推荐附近的用户
- **话题广场**：热门话题、最新话题、关注话题
- **内容分类**：按分类浏览不同类型的内容
- **搜索功能**：搜索用户、动态、话题

**操作指南**：
1. **切换标签**：点击顶部标签切换不同内容
2. **查看详情**：点击内容卡片查看详细信息
3. **关注用户**：点击用户卡片上的关注按钮
4. **参与话题**：点击话题卡片进入话题详情页
5. **搜索内容**：在搜索框中输入关键词进行搜索

### 3. 消息页

**功能说明**：查看系统通知、互动消息、聊天消息等。

**核心功能**：
- **系统通知**：系统公告、活动通知、违规提醒
- **互动消息**：点赞、评论、关注、提及通知
- **聊天消息**：一对一聊天、群聊消息
- **消息分类**：按类型分类显示消息
- **消息已读**：标记消息为已读

**操作指南**：
1. **切换消息类型**：点击顶部标签切换消息类型
2. **查看消息详情**：点击消息卡片查看详细信息
3. **回复消息**：点击消息卡片进入回复界面
4. **标记已读**：点击消息卡片上的已读按钮
5. **删除消息**：长按消息卡片，选择删除

### 4. 聊天页

**功能说明**：与好友进行实时聊天，支持多种消息类型。

**核心功能**：
- **好友列表**：显示最近聊天的好友
- **实时聊天**：发送文字、表情、图片、视频等
- **消息已读**：显示消息已读状态
- **消息撤回**：撤回已发送的消息
- **聊天记录**：查看历史聊天记录
- **群聊功能**：创建和管理群聊

**操作指南**：
1. **选择好友**：在好友列表中点击好友头像
2. **发送消息**：在输入框中输入消息，点击发送按钮
3. **发送图片**：点击图片图标，选择图片发送
4. **发送表情**：点击表情图标，选择表情发送
5. **撤回消息**：长按已发送的消息，选择撤回

### 5. 个人中心

**功能说明**：管理个人资料、查看个人动态、设置个人偏好。

**核心功能**：
- **个人资料**：编辑头像、昵称、性别、生日、简介等
- **个人动态**：查看和管理自己发布的动态
- **关注/粉丝**：查看关注的人和粉丝
- **收藏内容**：查看和管理收藏的内容
- **个人设置**：账号安全、隐私设置、通知设置等
- **等级与积分**：查看个人等级和积分

**操作指南**：
1. **编辑资料**：点击个人资料编辑按钮
2. **查看动态**：在个人中心滑动查看自己的动态
3. **查看关注**：点击关注数查看关注列表
4. **查看粉丝**：点击粉丝数查看粉丝列表
5. **进入设置**：点击设置按钮进入设置页面

### 6. 发布功能

**功能说明**：发布文字、图片、视频等多种形式的内容。

**核心功能**：
- **多类型发布**：支持文字、图片、视频
- **话题标签**：添加话题标签，增加内容曝光
- **地理位置**：添加发布位置
- **隐私设置**：设置内容可见范围
- **草稿保存**：保存未完成的内容为草稿

**操作指南**：
1. **点击发布按钮**：在首页或个人中心点击发布按钮
2. **选择内容类型**：选择文字、图片、视频
3. **编辑内容**：输入文字，添加图片或视频
4. **添加话题**：输入 #话题名 添加话题标签
5. **设置隐私**：选择内容可见范围
6. **点击发布**：完成内容编辑后点击发布按钮

### 7. 搜索功能

**功能说明**：搜索用户、动态、话题等内容。

**核心功能**：
- **全局搜索**：搜索所有类型的内容
- **分类搜索**：按用户、动态、话题分类搜索
- **搜索历史**：记录最近搜索的关键词
- **热门搜索**：显示热门搜索关键词
- **搜索建议**：根据输入自动提示搜索建议

**操作指南**：
1. **点击搜索框**：在顶部导航栏点击搜索框
2. **输入关键词**：输入要搜索的关键词
3. **选择搜索类型**：选择搜索用户、动态或话题
4. **查看搜索结果**：点击搜索按钮查看搜索结果
5. **清除搜索历史**：在搜索历史页面点击清除按钮

### 8. 话题功能

**功能说明**：创建、关注、参与话题讨论。

**核心功能**：
- **话题详情**：查看话题介绍、参与人数、相关动态
- **话题关注**：关注感兴趣的话题
- **话题发布**：在话题下发布相关动态
- **热门话题**：查看当前热门话题
- **话题分类**：按分类浏览话题

**操作指南**：
1. **点击话题标签**：在动态中点击话题标签进入话题详情页
2. **关注话题**：在话题详情页点击关注按钮
3. **发布话题动态**：在话题详情页点击发布按钮
4. **浏览话题动态**：在话题详情页滑动查看相关动态

### 9. 通知功能

**功能说明**：接收系统通知、互动通知、消息通知。

**核心功能**：
- **通知分类**：系统通知、互动通知、聊天通知
- **通知提醒**：新通知红点提醒
- **通知详情**：查看通知详细信息
- **通知已读**：标记通知为已读
- **通知设置**：设置通知接收偏好

**操作指南**：
1. **查看通知**：点击顶部导航栏的通知图标
2. **切换通知类型**：在通知页面点击顶部标签
3. **查看通知详情**：点击通知卡片查看详细信息
4. **标记已读**：点击通知卡片上的已读按钮
5. **设置通知**：在个人设置中设置通知偏好

### 10. 系统设置

**功能说明**：管理账号安全、隐私设置、通用设置等。

**核心功能**：
- **账号安全**：修改密码、绑定手机、绑定邮箱
- **隐私设置**：设置谁可以看我的动态、谁可以评论我
- **通知设置**：设置接收哪些类型的通知
- **通用设置**：清除缓存、深色模式、语言设置
- **关于我们**：查看版本信息、用户协议、隐私政策

**操作指南**：
1. **进入设置**：在个人中心点击设置按钮
2. **选择设置项**：点击需要修改的设置项
3. **修改设置**：根据提示修改设置内容
4. **保存设置**：点击保存按钮

## 常见问题

### 1. 登录问题

**问题**：无法登录账号
**解决方案**：
- 检查用户名和密码是否正确
- 检查网络连接是否正常
- 尝试使用验证码登录
- 点击忘记密码重置密码

**问题**：登录后无法加载内容
**解决方案**：
- 检查网络连接是否正常
- 清除浏览器缓存
- 刷新页面重试
- 尝试使用其他浏览器

### 2. 发布问题

**问题**：无法发布动态
**解决方案**：
- 检查网络连接是否正常
- 检查内容是否符合规范
- 检查图片或视频大小是否超过限制
- 尝试重新登录后发布

**问题**：发布的内容不显示
**解决方案**：
- 检查内容是否需要审核
- 检查网络连接是否正常
- 刷新页面后查看
- 联系客服处理

### 3. 互动问题

**问题**：无法点赞或评论
**解决方案**：
- 检查网络连接是否正常
- 检查账号是否被限制
- 刷新页面后重试
- 尝试重新登录后操作

**问题**：消息发送失败
**解决方案**：
- 检查网络连接是否正常
- 检查对方是否已将你拉黑
- 刷新页面后重试
- 尝试重新登录后发送

### 4. 性能问题

**问题**：页面加载缓慢
**解决方案**：
- 检查网络连接是否正常
- 清除浏览器缓存
- 关闭其他占用网络的应用
- 尝试使用更快的网络

**问题**：应用卡顿
**解决方案**：
- 清除应用缓存
- 关闭后台运行的应用
- 检查设备存储空间是否充足
- 尝试重启设备

### 5. 功能问题

**问题**：某些功能无法使用
**解决方案**：
- 检查账号权限是否足够
- 检查应用是否为最新版本
- 尝试重新登录后使用
- 联系客服处理

**问题**：地理位置服务无法使用
**解决方案**：
- 检查设备定位是否开启
- 检查应用是否有权限获取位置
- 尝试在设置中开启定位权限
- 重启设备后重试
